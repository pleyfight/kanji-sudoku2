# Architecture Decision Records (ADR)

This folder stores architecture decisions that are expected to outlive
individual implementation details.

## Conventions

- File format: `NNNN-short-title.md`
- Numbering: ascending, zero-padded (`0001`, `0002`, ...)
- Status values: `Proposed`, `Accepted`, `Superseded`
- Keep each ADR focused on one decision.

## Index

- `0000-template.md` - ADR template
- `0001-web-platform-scope.md` - Web-only platform scope
- `0002-ci-quality-gate.md` - Enforceable CI quality gate
- `0003-code-splitting-lazy-loading.md` - Code splitting and lazy loading
